# acquy-hungthinh-server

For acquyhungthinh server

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/anhpd912/acquy-hungthinh-server.git
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the server, run one of the following commands:

- For production:
  ```bash
  npm start
  ```
- For development with nodemon:
  ```bash
  npm run dev
  ```
